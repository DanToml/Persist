StoreType

public enum StoreType

StoreType is a simple encapsulation of the state required to setup the store for a Core Data stack.

  • InMemory: The InMemory store type is great for test enironments, or for providing simple in-memory caches with great querying.

  • SQLite: The SQLLite store is equivalent to NSSQLiteStoreType. This is the reccomended store type to use for most situations.

  • Binary: The Binary store is equivalent to NSBinaryStoreType.

  • Custom: Provide a custom NS*Store type.
